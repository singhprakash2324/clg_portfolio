# Prakash Singh — Professional Portfolio

A sleek, modern, and high-performance professional portfolio website built with a focus on aesthetics and user experience.

![Portfolio Preview](profile-pic.png)

## 🌟 Features

- **Professional Design**: A premium "Midnight & Emerald" color scheme designed to wow visitors.
- **Responsive Layout**: Fully responsive design that looks great on desktops, tablets, and mobile devices.
- **Dark/Light Mode**: Integrated theme toggle with smooth transitions and persistent settings.
- **Dynamic Animations**: Smooth reveal-on-scroll effects, magnetic buttons, and glassmorphism elements.
- **Interactive Components**: Including skill bars, activity charts (SVG), and infinite project/certification sliders.
- **SEO Optimized**: Semantic HTML5 structure with optimized meta tags.

## 🛠️ Technologies Used

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Libraries**: 
  - [Remix Icon](https://remixicon.com/) for sleek iconography.
  - [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/) for premium 3D tilt effects.
  - [Google Fonts](https://fonts.google.com/) (Inter & Plus Jakarta Sans).
- **Automation**: Minimal Python utility for UI updates.

## 📂 Project Structure

- `index.html`: Main structure and content.
- `style.css`: All styling, including the custom design system and theme variables.
- `script.js`: Interactive logic, animations, and theme management.
- `assets/`: Project images and icons.
- `resume/`: CV and resume assets.
